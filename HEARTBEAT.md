# CTO Heartbeat — Recurring Tasks

## Daily Research Cycle (06:00 UTC)
Run the full research methodology:
1. Scan all sources (GitHub Trending, HN, arXiv, YouTube AI channels, product changelogs, HuggingFace, Reddit r/MachineLearning)
2. Score each finding 0-10 against my interests
3. Deduplicate cross-platform
4. For items scoring 7+: enrich with summary, comparison to current stack, material assessment
5. Make autonomous decisions (adopt/reject/defer) per decision-evaluate skill
6. Log all decisions to logs/decisions/
7. Send daily digest to Telegram
8. Archive findings to knowledge base
9. Update wiki pages if findings change existing knowledge

## Health Checks (Every 30 Minutes)
- LLM API reachable
- Telegram bot responsive
- Memory/wiki readable
- VPS disk space >10% free
- RAM <90% used

## Weekly
- Review technology radar — any deferred items due for re-evaluation?
- Review MEMORY.md — anything stale that should be updated or removed?
- Review scoring calibration — am I filtering too loose or too tight?
